Detailed Description

Code serveur réseau réalisé en python, nous y retrouverons toutes les fonctions permettant d'écouter de communiquer avec le client ainsi que la base de données.

Author
VOLQUARDSEN Alex VAYSSE Matthieu
Version
1.0.5
Since
31/10/2022

Format des message normalement reçu par le serveur :

verification éligibilité pour ouvrir ou fermer un casier : [id_utilisateur],[id_casier],[id_carte] reservation d'un casier : [id_utilisateur],[id_casier] -> si le casier est déjà réserver, ou qu'il est occupé, ou que l'utilisateur ne peut pas réserver de casier, alors retourne une erreur fermeture d'un casier : [id_utilisateur],[id_casier],[commande(ici fermeture)] -> si l'utilisateur peut occuper le casier et que le casier est libre alors il se ferme et est attribuer à l'utilisateur ouverture d'un casier : [id_utilisateur],[id_casier],[commande(ici ouverture)] -> si l'utilisateur utilise bien se casier actuellement, alors il s'ouvre sinon erreur
